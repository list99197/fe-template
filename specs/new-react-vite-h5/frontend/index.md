# Frontend Development Guidelines

> Best practices for frontend development in **React + Vite H5** projects.

---

## Overview

This directory contains guidelines for frontend development. It follows the same index layout as the reference `frontend` spec folder; fill in each file with your project's specific conventions.

**Stack context**: React, Vite, mobile-first H5 (viewport, touch, performance on mid/low-end devices).

## How to Fill These Guidelines

For each guideline file:

1. Document your project's **actual conventions** (not ideals)
2. Include **code examples** from your codebase
3. List **forbidden patterns** and why
4. Add **common mistakes** your team has made

The goal is to help AI assistants and new team members understand how YOUR project works.
