| 目录 | 用途 |
|------|------|
| `src/components` | 跨页面复用的 **纯 UI / 复合业务无关** 或 **轻业务** 组件 |
| `src/layouts/components` | 与 **布局骨架** 强相关的区块（顶栏、侧栏菜单项、多栏容器等） |

## 选型指引（对照需求原型）

| 原型元素 | 建议优先组件 / 能力 |
|----------|---------------------|
| 带分页或「加载更多」的列表 | 使用支持 **分页器** 或 **无限滚动** 的表格/列表封装（如 **CommonTable** 对应模式） |
| 需固定列、横向滚动的大量字段 | 选用已支持 **sticky 列 / 横向滚动容器** 的表格方案 |
| 批量选择、批量操作 | 选用带 **行选择列**、**toolbar 批量操作区** 的封装 |
| 仅简单静态展示、无交互 | 可用轻量列表或文档约定的 List 组件；仍优先查表内是否有 **Empty / Skeleton** 统一态 |
